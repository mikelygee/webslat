WebSLAT is an on-line tool for performing seismic loss assessment, using the OpenSLAT library provided by the Quake Centre.

The WebSLAT code is available on github.

For support or further information, contact openslat@gmail.com

  • The work of University of Canterbury Professor Brendon Bradley
  • FEMA P-58 (Seismic Performance Assessment of Buildings) (SLAT uses the "Simplified Analysis" described starting on page 5-14).
